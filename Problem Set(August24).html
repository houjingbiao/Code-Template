1. driver and busman

在公共汽车上，为了保证乘客的安全，司机和售票员应该协调工作：停车后才能开门，关车门后才能行车。用PV操作来实现他们之间的协调。

semaphore s1 = 1;//是否允许driver启动汽车
semephore s2 = 1;//是否允许busman开门
driver(){
	while(true){
		P(s1);
			start the bus;
			running;
			stop the bus;//停车之后并不一定要V(s1)
		V(s2);
	}
}
busman(){
	while(true){
		close door;//循环的开始，反正是循环，从哪里开始呢？

	}
